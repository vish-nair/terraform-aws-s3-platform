# cross-account-bucket

Tier-2 app module. Extends `simple-bucket` with a cross-account bucket policy statement granting a role or account in a different AWS account scoped `GetObject`, `PutObject`, and `ListBucket` access. Provide the full ARN of the cross-account principal in `cross_account_role_arn`.
