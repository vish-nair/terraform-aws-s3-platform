# replication

Configures S3 cross-region or cross-account replication on an existing versioned bucket. Supports multiple replication rules (one per destination), Replication Time Control (RTC), and cross-account ownership override. Both `enable_versioning` and `enable_replication` must be `true` for the replication configuration to be created.
